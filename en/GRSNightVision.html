GRS Night Vision and Lights - Project Status Report

Date: January 8, 2026
Platform: Arma Reforger (Enfusion Engine)
Developer: Lucalis / GRS Modding Team

Executive Summary

The GRS Night Vision and Lights mod is a comprehensive night vision, thermal imaging, and IR lighting system for Arma Reforger. The project provides an alternative/complement to the existing NV-System mod with additional features including HDR thermal imaging, IR strobes, weapon-mounted IR lasers, and cross-mod compatibility.

Current Status: Active Development - Core systems functional, some issues pending resolution.

System Architecture

Core Components

Multiplayer Architecture

State Replication

GRS_NVL_NetworkComponent
├── Flashlight State (bool) → All Clients
├── IR State (bool) → All Clients  
├── Strobe State (bool) → All Clients
├── NVG Active (bool) → All Clients
└── Thermal Active (bool) → All Clients

Authority Model

  • Client: Requests state changes via RPC
  • Server: Validates and broadcasts state
  • All Clients: Apply visual effects locally based on received state

Conclusion

The GRS Night Vision and Lights mod provides a robust alternative night vision system for Arma Reforger with unique features not found in other mods. The cross-mod compatibility system ensures players can use GRS IR devices with any NVG mod. The main outstanding issue is the first-person device hiding which requires further investigation into prefab and material configuration differences.

Overall Completion: ~90%

Blockers: Alpha bias first-person hiding not working consistently across all prefabs.
